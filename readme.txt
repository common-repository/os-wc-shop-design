OS WC Shop Design is giving some additional features and effects to the default woocommerce shop.
== Description ==

OS WC Shop Design is giving some additional features and effects to the default woocommerce shop. It will give a new look and feel for Shop page with a sliding effect on the Produt images and advanced product sorting option.

== Features ==
* WooCommerce 3 compatible
* Admin interface to manage WooCommerce shop appearance.
* Easy to Change the font family, "Add to Cart" text, content and title color etc.
* Variety of WooCommerce shop appearance options.
* Automatic product slider.


Support

So that others can share in the answer, please submit your support requests through the WordPress forums for OS WC Shop Design(http://wordpress.org/support/plugin/os-wc-shop-design).

== Installation ==

Minimum requirements:    
    
 Wordpress 4.3+   
 WooCommerce 2.6+ 
 PHP 5.4.x    
 MySQL 5.4.x+  

Installation from within backend:

1. In plugin manager, click "Add New".
2. Search for "os-wc-shop-design".
3. Click "install".
4. Activate the plugin once it is installed.
5. Click on "Design Setting" sub menu under WooCommerce menu, finish the installation and manage shop page appearance.

FTP upload installation method:
1. Upload the "os-wc-shop-design" folder to the "/wp-content/plugins/" directory using your FTP client.
2. Activate the plugin through the "Plugins" menu.
3. Go to the "Design Setting" sub menu under WooCommerce menu, finish the installation and manage shop page appearance.

Zip upload installation method (make sure uploads up to 5MB are allowed for your hosting):

1. Login to your WordPress site administrator panel and head over the "Plugins" menu.
2. Click "Add New".  
3. Choose the "Upload" option.
4. Click **Choose file** (**Browse**) and select the os-wc-shop-design.*.zip file.   
5. Click **Install Now** button.    
6. Once it is complete, activate the plugin.   
7. Go to the "Design Setting" sub menu under WooCommerce menu, finish the installation and manage shop page appearance.
